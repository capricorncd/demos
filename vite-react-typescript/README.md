# vite-react-typescript

Vite, React(Hook), TypeScript

## Dev

### Setup

```shell script
npm i
```

### run dev

```shell script
# development
npm run dev
# > Local:    http://localhost:3000/
# > Network:  http://*.*.*.*:3000/
# yarn dev
```

### build

```shell script
# build
npm run build
```

### server

```shell script
# serve
npm run serve
```

## Generation Docs

执行一下命令，将读取`src/types`里的文件生成相应的`Markdown`文档文件。

```shell script
# generation docs
npm run docs
```
